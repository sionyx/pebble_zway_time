# ZWay Time
ZWay Time for Pebble - wearable client for RaZberry

ZWay Time application allows you to control your RaZberry based smart home. 
Devices are grouped into locations.

* on-off switches;
* motion sensors;
* humidity sensors;
* light sensors;
* temperature sensors.
